# proj-edge-ai-app

# Requirement
## nodejs
- 8.x
## NPM(Node Pacakge Manager)
- v5.6 or higher
- yarn
- grunt

# pre installed
```
# yarn명령어가 실행되지 않은 경우 아래의 명령을 실행해 주세요.
sudo npm install -g yarn
# grunt 명령어가 실행되지 않은 경우 아래의 명령을 실행해 주세요.
sudo npm install grunt-cli
```
# Build
```
yarn install
grunt
```
